CORREÇÃO PARA GITHUB PAGES

1. Extraia o ZIP.
2. Substitua TODOS os arquivos do repositório pelos arquivos desta pasta.
   Não envie apenas index.html.
3. Faça commit e push.
4. Aguarde o GitHub Pages concluir o deployment.
5. Abra o site e use Ctrl+F5.

O pacote usa parâmetros ?v=20260720-2 nos arquivos CSS e JavaScript para
forçar o navegador/GitHub Pages a buscar a versão atualizada.

Arquivos especialmente importantes para a nova home:
- index.html
- style.css
- home.js
- data.js
- core.js
